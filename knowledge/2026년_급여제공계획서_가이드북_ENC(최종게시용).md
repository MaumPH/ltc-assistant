⑤ 공단과 사례회의 결과 급여제공계획서의 변경이 필요한 경우
⑥ 욕구조사 결과 반영을 위해 급여제공계획서의 변경이 필요한 경우

# 개인별장기요양이용계획서 열람신청

- ○ 개인별장기요양이용계획서 열람신청 화면경로: 급여계약통보서/급여제공계획서/개인별장기요양이용계획서 열람신청 조회화면: 급여계약통보서/급여제공계획서/급여제공계획서
- - 통보한 급여제공계획서 내역을 바탕으로 급여제공계획서 모니터링 등록 가능
 개인별장기요양이용계획서 열람신청 내역 조회
‚ 구분: 신청 상태에 따라서 신청, 신청서(서식출력), 열람취소 할 수 있는 버튼 생성
- - 신청: 개인별장기요양이용계획서를 열람하기 위한 신청
- - 신청서: 열람신청 이후 개인별장기요양이용계획서 열람신청서 확인(미리보기 및 출력)
- - 신청취소: 처리상태가 ‘접수’인 경우, 착오신청 등 사유로 열람신청 내역 자체를 취소할 때 사용
- - 열람취소: 처리상태가 ‘완료’인 경우, 수급자 취소, 착오신청, 기타 사유로 열람을 취소할 때 사용
※공단에서 내역 확인 후 처리되면 열람 불가능
ƒ 처리상태: 접수, 완료 상태로 구분하여 각 신청 내역별로 처리 상태를 확인 할 수 있음
- - 접수: 기관에서 열람 신청한 상태
- - 완료: 기관에서 신청하고, 공단에서 승인이 완료된 상태
„ 동의여부: 개인별장기요양이용계획서 열람신청이 완료된 경우 Y로 표시, 열람신청 내역 없는 경우 N표시
 급여제공계획서 화면 바로가기
- 수급자 선택: 선택한 수급자의 급여제공계획서 화면으로 이동
- 수급자 미선택: 급여제공계획서 화면으로 이동
 급여제공계획서 내역 화면에서 열람동의여부, 열람가능여부 모두 Y로 표시되어 있는 수급자 선택
‚ 개인별장기요양이용계획서 조회 버튼 클릭
ƒ ‘개인별장기요양이용계획서 조회’팝업에서 열람할 계획서 클릭 시 조회ž출력 가능
※ 열람동의여부와 열람가능여부 중 한 항목이라도 ‘N’으로 표시될 경우 조회 불가능
Ÿ 공단에서 접수 및 승인 완료 후 기관에서 열람 가능 (처리기한: 접수일로부터 7일 이내(접수일 포함, 토/공휴일 미포함))
