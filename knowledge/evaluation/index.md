# 2026 주야간보호 평가지표 인덱스

이 파일은 평가지표 탭의 인덱스입니다.  
각 영역별 지표 파일을 `knowledge/evaluation/` 폴더에 추가하면 좌측 탐색 메뉴에 자동으로 표시됩니다.

## 파일 작성 방법

각 지표 파일은 아래 형식으로 작성하세요:

```
---
title: 1-1. 지표명
area: 1영역: 기관 운영
status: active
updated: 2026-04-10
tags: [인력, 배치기준]
---

## 판단 기준
...

## 확정 근거
- 문서명, 조문, 시행일

## 실무 해석
...

## 주의사항
...
```

## status 값 설명

| 값 | 의미 |
|----|------|
| `active` | 현행 기준 |
| `revised` | 개정됨 (내용 확인 필요) |
| `확인필요` | 불명확 또는 검토 중 |
